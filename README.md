# CodeAlpha Data Science Internship Projects

This repository contains the machine learning and data analysis projects completed as part of the CodeAlpha Data Science Internship.

## Projects

### 1. Iris Flower Classification

* Classification of Iris flower species using machine learning.
* Techniques used: Data preprocessing, exploratory data analysis, Random Forest Classification, model evaluation, and prediction.

### 2. Car Price Prediction

* Prediction of used car selling prices based on vehicle characteristics.
* Techniques used: Feature engineering, Random Forest Regression, model evaluation, and feature importance analysis.

### 3. Unemployment Analysis with Python

* Analysis of unemployment trends using data visualization and exploratory data analysis.
* Focus areas: Regional unemployment patterns, labour participation rates, employment distribution, and COVID-19 impact.

### 4. Sales Prediction Using Python

* Prediction of product sales based on advertising expenditure across TV, Radio, and Newspaper channels.
* Techniques used: Linear Regression, correlation analysis, data visualization, and model evaluation.
